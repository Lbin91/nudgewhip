## FAQ

### Q1. What data does Nudge collect?
Nudge does not collect keystroke content, screen content, or browsing history. It only uses whether global input activity is occurring (active/idle). All data is stored locally on your device.

### Q2. Why does Nudge need Accessibility permission?
macOS requires Accessibility permission to detect global input activity. Nudge uses this permission solely for input activity detection and does not access keystroke content or screen content.

### Q3. What is the difference between Free and Pro?
Free covers the Mac-only return loop. Pro adds iPhone follow-up (remote escalation), break mode, whitelist exception controls, and daily stats.

### Q4. Is iPhone follow-up real-time?
No. It uses CloudKit-based best-effort near real-time sync. Delivery may be delayed depending on network conditions. Real-time delivery is not guaranteed.
