# Nudge Copywriting

- Version: 0.1
- Last Updated: 2026-04-02
- Owner: `marketing-strategist`
- Scope: launch website, app store, social, email, and waitlist copy

## 1. Voice

- 짧다
- 직접적이다
- 죄책감을 유발하지 않는다
- 프라이버시와 복귀 유도를 동시에 말한다
- `attention recall tool` 포지셔닝을 흐리지 않는다

## 2. Core Message

- KR: 딴짓을 막는 앱이 아니라, 딴짓이 시작된 순간 다시 돌아오게 하는 앱
- EN: Not a blocker. A return-to-work nudge the moment attention drifts.

## 3. Hero Copy

| Surface | KR | EN |
|---|---|---|
| Hero headline | 딴짓이 시작되면, 바로 돌아오게 | The moment attention drifts, Nudge brings you back |
| Hero subhead | 입력이 멈춘 순간을 감지해 부드럽게 작업 복귀를 유도하는 macOS 메뉴바 앱 | A macOS menu bar app that detects idle moments and gently brings you back to work |
| Supporting line | 키 입력 내용은 보지 않고, 전역 입력 활동만 사용합니다 | Nudge does not read typed text. It only uses global input activity |

## 4. CTA Copy

| Surface | KR | EN |
|---|---|---|
| Primary CTA | Waitlist 등록 | Join the waitlist |
| Secondary CTA | GitHub 보기 | View on GitHub |
| Tertiary CTA | iPhone 알림 소식 받기 | Get iPhone alert updates |
| Upgrade CTA | Pro로 업그레이드 | Upgrade to Pro |

## 5. App Store Copy

### 5.1 Short Description

- KR: 입력이 멈춘 순간을 감지해 부드럽게 작업으로 돌아오게 하는 macOS 메뉴바 앱
- EN: A macOS menu bar app that detects idle moments and gently brings you back to work

### 5.2 Feature Bullets

- KR: 전역 입력 기반 idle detection
- EN: Global input-based idle detection
- KR: Accessibility 권한 이유를 명확히 안내
- EN: Clear permission rationale for Accessibility
- KR: Free는 Mac 단일 복귀 루프
- EN: Free covers the Mac-only return loop
- KR: Pro는 iPhone 연동과 예외 제어 제공
- EN: Pro adds iPhone follow-up and exception controls

### 5.3 Review Notes

- 실시간 보장이라는 표현을 사용하지 않는다
- 화면 캡처, 키 입력 내용, 브라우징 기록 수집을 암시하지 않는다
- Pro 기능은 CloudKit 기반 `best-effort near real-time`로 설명한다

## 6. Social Copy

### 6.1 Launch Post

- KR: Nudge는 딴짓을 막는 앱이 아닙니다. 입력이 멈춘 순간을 감지해, 바로 돌아오게 하는 macOS 메뉴바 앱입니다. 기다리던 beta가 열렸습니다.
- EN: Nudge is not a blocker. It detects idle moments and nudges you back to work, right from the macOS menu bar. Beta is open.

### 6.2 Feature Post

- KR: Accessibility 권한은 전역 입력 활동 감지에만 사용합니다. 키 입력 내용, 화면 내용은 수집하지 않습니다.
- EN: Accessibility is used only for global input activity detection. Nudge does not collect keystroke or screen content.

### 6.3 Pro Post

- KR: Pro는 Mac + iPhone 복귀 루프, 휴식 모드, 화이트리스트, 상세 통계를 제공합니다.
- EN: Pro adds the Mac + iPhone return loop, break mode, whitelist controls, and detailed stats.

## 7. Email Copy

### 7.1 Waitlist Confirmation

- KR: Nudge 대기자 명단에 등록되었습니다. beta가 열리면 가장 먼저 알려드리겠습니다.
- EN: You are on the Nudge waitlist. We will let you know first when beta opens.

### 7.2 Beta Invitation

- KR: Nudge beta를 먼저 써보실 수 있습니다. Mac 메뉴바에서 바로 동작하고, 전역 입력 활동만 사용합니다.
- EN: You can try the Nudge beta first. It lives in the Mac menu bar and uses only global input activity.

### 7.3 Pro Launch

- KR: Pro가 열렸습니다. iPhone 연동, 예외 제어, 상세 통계를 한 번에 사용할 수 있습니다.
- EN: Pro is live. Get iPhone follow-up, exception controls, and detailed stats in one place.

## 9. Reuse Rules

- hero, web CTA, email, and app store copy는 같은 핵심 문장을 공유한다
- KR/EN은 의미를 맞추되 직역에 묶이지 않는다
- 길이가 길어질 경우 부가 설명을 빼고 핵심 가치만 남긴다
