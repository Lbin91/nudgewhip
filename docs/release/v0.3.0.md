# NudgeWhip v0.3.0

Cloud backup and overlay experience release.

This release adds CloudKit daily aggregate backup and improves the countdown overlay experience for longer sessions.

## Highlights

- CloudKit daily aggregate backup for cross-device data continuity
- Improved countdown overlay with mini-state feedback and hover dismissal
- Better overlay fatigue reduction for extended use

## What's New

### CloudKit Daily Aggregate Backup

NudgeWhip can now back up daily aggregate statistics to your iCloud Private Database.

This enables:

- daily focus duration, session count, alert count, and recovery metrics backed up to iCloud
- deterministic per-device, per-day records in a private CloudKit zone
- automatic upsert on session and recovery completion
- local-first architecture: CloudKit is a backup layer, not the source of truth
- graceful degradation when CloudKit is unavailable

What still does **not** happen:

- no raw input logs uploaded
- no session timeline or event-level detail uploaded
- no real-time sync guarantees
- no iOS companion fetch yet

### Improved Countdown Overlay

The countdown overlay now handles extended sessions better:

- mini overlay state with explanatory feedback when the overlay is dismissed
- hover-to-dismiss experiment support for power users
- reduced visual fatigue during long focus blocks

## Fixes

- fixed a crash when CloudKit container identifier was not configured in the build
- fixed UI test isolation so stored local settings no longer bleed between test runs

## Distribution Notes

`0.3.0` is distributed as a notarized GitHub release DMG for direct download on macOS.

This is a notarized Developer ID release, not a Mac App Store review release.

## Out Of Scope

- iOS companion app
- StoreKit / Pro paywall
- Pet progression
- Recovery Review dashboard
- Schedule presets
