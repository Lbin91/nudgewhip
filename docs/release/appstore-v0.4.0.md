# App Store Connect Metadata Draft — v0.4.0

## 앱 설명 (한국어)

NudgeWhip은 Mac에서 사용자의 비활성 상태를 감지하고, 작업 흐름이 끊어졌을 때 가볍고 즉각적인 넛지로 다시 집중할 수 있도록 돕는 메뉴 막대 앱입니다.

키보드 입력 내용이나 화면 내용을 수집하지 않고, 접근성 권한을 통해 마우스/키보드 활동 여부만 확인해 유휴 시간을 판단합니다. 사용자는 자신의 작업 스타일에 맞게 유휴 감지 시간, 알림 강도, 일정, 오버레이 표시 방식을 조절할 수 있습니다.

v0.4.0에서는 iPhone companion 연동을 통해 Mac의 현재 상태와 복귀 이력, 일간 통계를 CloudKit 기반으로 동기화할 수 있습니다. 여러 대의 Mac을 같은 iCloud 계정으로 사용하는 경우에도 기기별로 상태를 분리해 관리합니다.

주요 기능:

- Mac 유휴 상태 감지 및 단계별 넛지
- 일정 기반 모니터링 on/off
- 집중 복귀 패턴과 일간 통계 확인
- 화면 위 카운트다운 오버레이
- CloudKit 기반 iPhone companion 동기화
- 한국어/영어 로컬라이징

NudgeWhip은 강한 차단보다 가벼운 상기와 반복 가능한 루틴에 초점을 둡니다. 작업 리듬을 깨지 않으면서, 스스로 다시 돌아올 수 있는 환경을 만드는 것이 목표입니다.

## What's New (한국어, 170자 이내)

iPhone companion 연동과 CloudKit 실시간 동기화를 추가했습니다. Mac 상태, 복귀 이력, 일간 통계를 iPhone에서 확인할 수 있고, 다중 Mac 분리와 한국어 현지화도 강화했습니다.
