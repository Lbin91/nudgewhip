## 4. 문장 부호 불일치 (Punctuation — 3개)

번역은 자연스럽지만, 영어 원문의 문장 끝 부호(?, !)와 한국어 번역의 부호가 다름.
한국어 표현상 자연스러운 선택이므로 **선택적 수정 권장** (우선순위 낮음).

| 키 | 영어 | 한국어 | 비고 |
|----|------|--------|------|
| `alert.message.gentle_nudge` | ...Let's get back into focus. | ...다시 집중 흐름으로 돌아가볼까요? | EN은 마침표, KO는 물음표. 한국어 "볼까요?"가 자연스러워 OK. |
| `onboarding.preview.sound.whip.button` | Preview Whip! | Whip! 사운드 듣기 | EN은 감탄부호, KO는 없음. "사운드 듣기"로 의미 전달 OK. |
| `settings.section.accessibility.recovery.title` | Need a guided recovery path? | 복구 경로를 다시 보고 싶다면 | EN은 물음표, KO는 종결 어미 없음. "보고 싶다면"이 자연스러워 OK. |

## 5. 펫 관련 키 (Pet-related — 38개)

v0.4.0에서 펫 UI/서비스를 제거했으나, 번역 키는 xcstrings에 남아 있음.
다음 버전에서 펫 기능이 복원되면 즉시 활용 가능하므로 **유지 권장**.

주요 키 그룹:
- `pet.stage.*` (5개): 성장 단계명 (알, 새끼, 어린이, 청년, 장년)
- `pet.detail.*` (13개): 펫 상세 뷰 텍스트
- `pet.notification.*` (2개): 레벨업 알림
- `pet.summary.*` (1개): 요약 카드
- `onboarding.setup.pet_mode.*` (3개): 온보딩 펫 설정
- `settings.section.pet.*` (4개): 설정 펫 섹션
- `menu.dropdown.value.pet_emotion.*` (4개): 감정 표현
- `menu.dropdown.value.pet_mode.*` (2개): 펫 표시 모드
- `settings.section.monitoring.pet_mode*` (2개): 설정 모니터링
